【制作物情報】
このプログラムは学習用に制作したものです。
三次元の座標情報からレイキャスティングによってレンダリングすることが出来ます。
オブジェクトの回転、カメラの移動、レンダリング品質を設定してレンダリングします。

制作目標：3DCGへの理解を深めるため、ライブラリを出来るだけ使わずに制作する。

使用言語：Python
使用ライブラリ:Tkinter(GUIの作成、ピクセルの描画)
　　　　　　 time（レンダリング時間の計測）

【使用方法】
①Pythonがインストールされているなら「main.py」、
　Pythonがインストールされてなければ「main.exe」を実行してください。
（exeファイルは単体で動作しますが、pyファイルはフォルダ構成が変わると実行できません）

②ウィンドウサイズを指定し、決定ボタンを押してください。

③レンダリングボタンを左クリック、もしくはEnterキーを押すとレンダリングが開始されます。

【設定項目】
※すべての項目は半角数字にのみ対応しています

『画質設定』
・走査線間隔 ＝ 指定した数字×n 行目のみレンダリングされます。
（数値が大きくなるほどレンダリング時間が短縮されます）
（0以下には設定できません）

・ピクセルサイズ ＝ 1ピクセルの大きさ
（数値が大きくなるほどレンダリング時間が短縮されます）
（0以下には設定できません）

『カメラ設定』
・X座標,Y座標,Z座標 ＝ カメラの位置座標
（-35～35の間に設定するとオブジェクトと重なる可能性があります）

・焦点距離 ＝ カメラの焦点距離
（数値が大きいと視野角が狭くなり、パースがゆるくなります）
（負の数字を入れると後ろを向きます）

・X軸回転,Y軸回転,Z軸回転 ＝ カメラのワールド回転

『オブジェクト』
・X軸回転,Y軸回転,Z軸回転 ＝ ワールド軸でオブジェクトを回転(度)
（負の数字や360以上の数字にも対応しています）
（X→Y→Zの順に計算されます）
